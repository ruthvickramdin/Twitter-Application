Web Information Systems - WIS Twitter Assignment

Description of files:
                
--php/lib/ - PHP classes to be included (or "required") within your PHP 
             script(s). These files are not intended to be modified, but 
             rather to be used as a library similar to wis-twitter.jar.
                    
--php/apidocs/ - Contains the PHPDoc for the PHP class libraries. See index.html.

If using Python, please refer to the README that describes how to set up your 
virtualenv, etc.
